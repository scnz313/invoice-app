## 🚀 Getting Started

### Prerequisites
- Flutter 3.0+ installed
- Dart 3.0+
- iOS Simulator / Android Emulator or Physical Device

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/scnz313/invoice-app.git
   cd invoice-app
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```
